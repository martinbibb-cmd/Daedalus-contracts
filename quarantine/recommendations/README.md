# Recommendation Contract Quarantine

Status: Quarantined

This directory preserves the former recommendation contract artifacts for migration reference only.

They are not part of the active Contracts source tree, are not exported from `src/index.ts`, are not included by `tsconfig.json`, and must not be treated as public Daedalus contracts.

Daedalus Contracts defines captured and declared reality. Recommendation, ranking, scoring, product-selection, quote-selection, cost-saving, and priority logic belongs outside Contracts.
